centos7编译libwebp

  • centos7 libwebp
Tuesday, December 31st 2019

下载源码包:
https://github.com/webmproject/libwebp/releases
libwebp-v1.0.3.zip

1.jepg、png、tiff插件安装

2.简单编译

仅仅编译examples/cwebp和examples/dwebp，与static library src/libwebp.a。
使用makefile.unix进行编译


成功：

3.完整编译

 yum install gcc make autoconf automake libtool
 cd libwebp-1.0.3/  
 #生成configure脚本
 ./autogen.sh
 ./configure
make
make install

会生成如下目录和文件：
/usr/local/include/webp/decode.h
/usr/local/include/webp/encode.h
/usr/local/include/webp/types.h
/usr/local/lib/libwebp.*
/usr/local/bin/cwebp
/usr/local/bin/dwebp

如需你需要生成libwebpdecoder， ./configure 需要带上选项：
–enable-libwebpdecoder
更多选项信息请使用以下命令查看：
./configure –help

4.测试

cwebp -q 70 test.jpg -o out.webp
